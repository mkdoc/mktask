> Run build tasks

Runs named task functions that return streams.

## Install

```
npm i mktask --save
```

For the command line interface install [mkdoc][] globally (`npm i -g mkdoc`).

## API

### Task

```javascript
Task()
```

Task runner entry point.

### task

```javascript
task()
```

Adds task function(s) to the list of known tasks.
